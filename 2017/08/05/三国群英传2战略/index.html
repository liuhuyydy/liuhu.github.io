三国群英传2战略

发表于 2017-08-05 | 分类于 游戏 |

1、少兵单挑：
主将带奇数的很少的兵，而敌将要带相对很多的兵。武将前列，开战后左右分散，分到比较靠边了，全军后退武将出击，这样敌军全军突击后，敌兵就跑去找你的兵，而你的主将可以直接面对敌将单挑获胜。这招很常用，适用范围有必要说下：不止适用于武力高的将，只要相对于敌将武力高，或者武力相近我方有强力杀将技或恢复技。举例：1.简雍对管恪，简雍武力高且有落月弓，必胜2.邢道荣对曹仁，武力即使稍低曹仁也敌不过连弩狂涛。

2、穿城大法：
敌人通常在与你接壤的城布置很多将，而后方就空虚，所以派一批人进攻前线敌将多的城进入战斗画面后撤兵，回到地图画面后立刻选中我方部队，这时可以变更这支部队的目的地到敌人后方，部队就穿过敌城进攻后面人少的城了，这个常用于堵后路，更可以用于加快行军速度（逃跑速度很快），记住撤退一次士气降12点，要保证真正开战时士气在51以上就行。

3、远程进城：
简单说：当同时选中我方部队与我方城池时，被选中部队可以有“进入城内”的选项出现，而电脑不会理会此时部队与城市的实际距离有多远。新手要学会这个技巧，可以自己去试着把部队从官渡调到界桥，最快的办法：部队从官渡出发，设为目的陈留，在跑到一半时点部队的旗子，这时界桥与部队一同出现，选部队进入城内，好，部队到界桥了。这对于加快行军很有用。

4、虐敌：
对于一支已经抓住的敌军，有杀招放三选项，我们一般招，招不来才杀或放。但是有个办法就是在敌军已经被我方重重包围时，直接放，然后不停的用我方的将把这些血空的敌人抓了放放了抓，每打一次有如下好处：1.我方将升级；2.我方全体士气+2，刚得到的没士气的大将也很快能带兵上阵；3.被抓将领所属的敌方士气降2，后面打起来就是哗哗哗的跑兵；4.可以用恢复体力技力的军师技配合伏兵回体武将技，在虐敌等待的漫长战斗过程中把一支刚打完仗，血或兵有残缺的部队完全恢复战斗力。这是需要时间相当长的一招，好处多的很，但是用多了有“名为虐敌实为自虐”的感觉，因为总是在打极其无聊的必胜战。最后说下，招降BUG我就不讲了，那个更无聊，都不用包围敌人就实现无限循环战。

5、控兵成诈败：带40左右的兵，武将前列，一开战，全军后退一点，中央集中，当集中到武将身后到屏幕最右边有一条完整的绿线时，全军待机。如此则成诈败之势，敌军无法围殴你的主将，敌将上来则完全形成单挑。在兵少时，利用这个既有诈败的效果，又可以使用其他恢复增加类好军师技。这个办法经常被敌将的杀兵技打乱，这样就要用到下一个技巧。

6、逃跑耗敌：在看到敌人一批较强的力量来进攻时，无法及时增援，弃城又不甘心，那就先派些人出去消耗，进入战斗后选人出击，如果我方集气快的，就放技耗他的兵，如果集气慢就把握好敌方的集气速度，然后立刻跑，敌将气满之后一般是立刻用技，一个技也无法打掉我的将，但是消耗了他的技力。一般耗两三次，敌人就全体无技力可用了。然后我方快速逃回城，还可以恢复以待决战。注意让不同的人出去耗，这样士气普遍降的不多。这个还可以用于带远程兵，射几轮既耗兵又耗他技，不过自己也有损兵的可能。

7、出城迎敌：城市被打一次就减人口，开发可不容易，所以在有胜利把握时，看见敌人来战则必然出城迎敌，即使抓逃，也要有出城打的习惯，城市开发度是兵的重要来源，不可轻易降低。

8、主动求败：这个算用的很少，比如有个我想要的强人在打到还剩几十血就逃跑，这时没兵围他，追击成功率太低，而我处于靠后的位置，那就我也逃跑，比他先跑出屏幕就算我败，下一仗找个好用的抓他。最好就是用于吕布魏延之流，基本上抓到即归我。
